# PayPaddy

Nigerian escrow fintech — Universal escrow with four trust moats.

## Getting started

```bash
cd app
npm install
npx expo start
```

See [`app/README.md`](app/README.md) for the full developer guide, stack
overview, route map, domain model, and verification checklist.
